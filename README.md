
## Laravel and Vue

Laravel and Vue Admin panel:

### Setup Commands

- **composer install**
- **npm install**

### Setup Database
- **create new database**
- **copy env.example and remove env**
- **change database name into env**
- **php artisan key:generate**
- **php artisan migrate**
- **php artisan db:seed**

### Start Commands

- **php artisan serve**
- **npm run watch**
